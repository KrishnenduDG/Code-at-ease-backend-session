# Backend Basics Session

### How to Run the Server ?

- Clone it

```bash
git clone https://github.com/KrishnenduDG/Code-at-ease-backend-session
```

- Navigate to the directory

```bash
cd Code-at-ease-backend-session
```

- Install the Dependencies

```bash
npm i
```

- Set the Environment Variables in `.env` file (just setting the `PORT` variable will suffice for this)

- Run the server

```bash
npm run dev
```
